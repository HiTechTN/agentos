Serveur de Production

Un VPS ou serveur dédié avec Docker ≥ 24 et Docker Compose ≥ 2.0. Configuration minimale : 2 CPU, 4 Go RAM, 20 Go SSD.

💡 L'adresse IP publique ou le nom de domaine de votre serveur
DigitalOcean Hetzner Cloud AWS EC2 GCP Compute Documentation GitHub Secrets
💡 Utilisateur avec droits sudo et clé SSH autorisée
adduser deploy && usermod -aG sudo deploy Guide de configuration initiale du serveur
🔑 La clé privée SSH qui correspond à la clé publique ajoutée au serveur
ssh-keygen -t ed25519 -C "deploy-agentos" -f ~/.ssh/agentos_deploy ssh-copy-id -i ~/.ssh/agentos_deploy.pub deploy@<IP_DU_SERVEUR> Documentation GitHub SSH
