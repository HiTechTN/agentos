API Reference v7.2.2

Complete REST API for AgentOS — 60+ endpoints across 12 resource groups

Base URL: https://your-host:8000 — All endpoints return APIResponse[T] envelope: {"data": ..., "meta": {...}, "errors": []}

CodeMeaning
200OK — Success
201Created — Resource created
202Accepted — Async operation started
400Bad Request — Validation error
401Unauthorized — Missing/invalid JWT
422Validation Error — Pydantic schema
429Rate Limit Exceeded — Too many requests
500Internal Server Error

🚀 Run & Plan

POST /api/v1/run JWT Execute an AI prompt

Main entry point. Routes the prompt to the appropriate agent(s) and executes the workflow with Intelligence Engine enrichment.

Request Body

promptstringrequiredNatural language task description
project_idstringoptionalProject/workspace identifier
agentstringoptionalForce specific agent (dev, content, marketing, commerce)

Rate Limit

10 requests per minute

bash
curl -X POST https://your-host:8000/api/v1/run \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{"prompt": "Create a REST API with FastAPI", "project_id": "my-project"}'
POST /api/v1/plan JWT Generate a structured plan

Uses @Planner sub-agent to produce a structured plan with phases, tasks, risks, and architecture.

Rate Limit

20 requests per minute

POST /api/v1/verify JWT Validate code/output

Uses @Verifier sub-agent to validate code quality, security, and correctness. Returns JSON issue report.

Rate Limit

15 requests per minute

🧠 Intelligence Engine

POST /api/v1/intelligence/memories JWT Record episodic memory

Request Body

workspace_idstringrequiredWorkspace identifier
task_typestringrequirede.g. code_gen, content_write
prompt_summarystringrequiredShort description
outcomestringrequiredsuccess | failure | partial
quality_scorefloatoptional0.0 - 1.0
GET /api/v1/intelligence/memories JWT Recall similar memories

Query Parameters

task_typestringoptionalFilter by task type
workspace_idstringoptionalFilter by workspace
limitintoptionalMax results (default 10)
outcome_filterstringoptionalsuccess | failure

🔌 WebSocket

WS /ws/logs Real-time agent logs
javascript
const ws = new WebSocket("ws://localhost:8000/ws/logs");
ws.onmessage = (event) => {
  const log = JSON.parse(event.data);
  console.log(`[${log.level}] ${log.agent_id}: ${log.action}`);
};
