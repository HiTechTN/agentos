🤖 Sub-Agent System

Planner

@Planner

auto-route: plan, design, architecture

Produces structured plans with phases, tasks, risks, dependencies, and architecture summaries from plain-language goals.

Verifier

@Verifier

auto-route: verify, validate, test

Validates code quality: lint, type safety, test coverage, security, edge cases. Returns JSON issue reports.

Explorer

@Explorer

auto-route: find, search, where is

Searches and analyzes codebases, traces dependencies, identifies relevant files, summarizes architecture.

Reviewer

@CodeReviewer

auto-route: review, audit, security

Senior engineer-level review: security analysis, performance, architecture, style, edge cases, error handling.

How it works

Clone & configure

git clone + cp .env.example .env — fill in your API keys.

Start the stack

docker compose up -d — 10 services spin up in seconds.

Create a plan

POST /api/v1/plan — @Planner produces structured plan with phases and tasks.

Execute & verify

Agents execute in parallel. @Verifier validates output. Kanban updates in real time.

Review & deploy

Approve HITL requests. Check Pulse dashboards. Deploy to production.
