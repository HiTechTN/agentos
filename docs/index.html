Features

✨ What's New in v7.0

The Intelligence Engine and 7 new modules transform AgentOS from an orchestrator into an autonomous learning system.

🧠

Intelligence Engine v6.0

Episodic memory, skills registry, knowledge base, self-reflection engine, and context enricher. The system learns from every execution and improves over time.

🌳

TreeOfThoughts v7.0

Multi-branch reasoning with up to 2 parallel thought paths. Activated automatically when task complexity > 0.7. Smarter decision-making for hard problems.

🔗

GraphRAG v7.0

Knowledge graph extraction and storage after every session. Entities and relationships are persisted to PostgreSQL for cross-session context retrieval.

🎮

VRAM Manager v7.0

Intelligent GPU memory manager for Ollama models. Ensures only one model in VRAM at a time (12GB budget). Auto-switches models based on work_type.

🔧

AutoCorrector v7.0

Wraps all sandbox DevAgent calls with automatic error correction. Detects failures, analyzes root cause, and retries with corrected code — zero manual intervention.

🚌

AgentBus v7.0

Event bus for spawning specialized sub-agents on demand. Enables dynamic agent composition — spawn, communicate, and retire agents at runtime.

⚡

WasmRunner v7.0

WebAssembly sandbox for lightweight tools (<5ms). Run untrusted code safely with near-native performance. Perfect for sandboxed DevAgent actions.

🖥️

ComputerUseTools v7.0

Desktop automation: mouse, keyboard, screenshot, clipboard. Disabled by default, activated via COMPUTER_USE_ENABLED=true. Requires 12GB GPU free.

🔗

MCP Integration

Model Context Protocol for extending AgentOS with any external tool server. Register databases, APIs, cloud services as MCP tools.

🔄

Episodic Memory

Records every task outcome with quality scoring. Recall similar past tasks to reuse successful strategies. Continuous improvement across sessions.

📚

Skills Registry

Auto-extracts reusable skills from successful task outcomes via LLM analysis. Reinforcement learning adjusts confidence. Skills auto-disable after 4 failures.

💡

Self-Reflection

Nightly cron job analyzes all task outcomes, detects patterns, generates recommendations, and evolves agent behavior. Configurable via nightly_reflection_cron.

🔄

Smart LLM Router

28+ free OpenRouter models across 8 work types. Auto-detection, rate-limiting (1 req/min sliding), 4-level fallback (OpenRouter → Ollama → degraded).

📊

Kanban + Pulse

6-column Kanban with WebSocket real-time updates. Pulse dashboards with latency charts, agent activity, task completion metrics.

🔐

Auth + Admin

Email/password + OAuth (Google/GitHub). JWT auth with role-based access. Admin panel: settings, services, LLM providers, user management.

📱

Cross-Platform UI

Next.js 16 web app + Tauri v2 desktop (deb/rpm/AppImage) + React Native mobile (APK) + PWA. Single React codebase, 4 platform targets.

Intelligence Engine

🧠 Learn. Adapt. Evolve.

The Intelligence Engine records every execution, extracts patterns, builds a knowledge graph, and reflects nightly — making AgentOS smarter over time.

Memory

Episodic Memory

app/memory/episodic.py

Records TaskOutcome with quality_score. Recall similar tasks by type, filter by outcome. Get best strategy for any task type.

Skills

Skills Registry

app/skills/registry.py

LLM auto-extracts skills from successes. Reinforcement learning (+0.05 success, -0.1 failure). Auto-disables after 4 consecutive failures.

Knowledge

Knowledge Base

app/memory/knowledge.py

6 knowledge kinds: api_pattern, code_pattern, constraint, best_practice, domain_fact, failure_mode. ILIKE search with confidence scoring.

Reflection

Self-Reflection

app/learning/reflection.py

Nightly analysis of all task outcomes. Detects patterns, generates recommendations, evolves agent behavior. Threshold: 10 tasks between reports.

Enricher

Context Enricher

app/learning/context_enricher.py

Augments system prompts with memories, skills, and knowledge before each execution. 3 optional blocks, capped at 2000 chars.

Graph

GraphRAG

app/memory/graph_rag.py

Entity-relation extraction after every session. Stores knowledge graph in PostgreSQL. Enables cross-session context retrieval.

Sub-Agent System

🤖 5 Specialized Sub-Agents

Auto-routed based on prompt content. Each sub-agent is an LLM-powered specialist with its own expertise and output format.

Planner

@Planner

auto-route: plan, design, architecture

Produces structured plans with phases, tasks, risks, dependencies, and architecture summaries from plain-language goals.

Verifier

@Verifier

auto-route: verify, validate, test

Validates code quality: lint, type safety, test coverage, security, edge cases. Returns JSON issue reports with severity levels.

Explorer

@Explorer

auto-route: find, search, where is

Searches and analyzes codebases, traces dependencies, identifies relevant files, summarizes architecture patterns.

Reviewer

@CodeReviewer

auto-route: review, audit, security

Senior engineer-level review: security (CRITICAL/HIGH/MEDIUM/LOW), performance, architecture, style, edge cases, error handling.

Debugger

@Debugger

auto-route: error, exception, debug, crash

Error analysis with root cause, explanation, fix suggestion, and confidence score. Integrates with AutoCorrector for automatic fix.

Domain Agents

4 specialized domain agents with parallel execution, retry (×3), and circuit breaker (3 failures → auto-disable).

💻

DevAgent

code

Scaffolds repos, configures CI/CD (GitHub Actions + GitLab CI), runs tests, deploys to staging. AutoCorrector wrapper for sandbox calls.

📝

ContentAgent

content

Writes SEO content, generates images via Replicate, manages editorial calendar. Multi-modal with vision-analysis skill.

📈

MarketingAgent

growth

Segments audiences, sends email campaigns (MailHog/SMTP), configures ads, generates performance reports with Pulse data.

💰

CommerceAgent

sales

Manages product catalogs, creates Stripe checkout sessions, tracks inventory, generates FAQ. Skills Registry optimizes strategies.

How it works

Clone & configure

git clone + cp .env.example .env — fill in your API keys.

Start the stack

docker compose up -d — 12 services spin up in seconds.

Create a plan

POST /api/v1/plan — @Planner produces structured plan with phases and tasks.

Execute & verify

Agents execute in parallel with AutoCorrector. @Verifier validates output. GraphRAG extracts knowledge.

Learn & evolve

Intelligence Engine records outcomes, updates skills, enriches context. Nightly self-reflection improves performance.

Review & deploy

Approve HITL requests. Check Pulse dashboards. Deploy to production.
