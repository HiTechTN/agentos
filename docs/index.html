Why AgentOS?

Built for founders, agencies, and teams who want to ship faster without the overhead.

🤖

Four specialized agents

Dev, Content, Marketing, and Commerce agents work together on your pipeline.

🛡️

Human-in-the-loop

Critical actions require your approval. Deployments, payments, and publications are gated.

🔄

LLM fallback

OpenRouter primary, Ollama local fallback, degraded response as last resort.

🧰

Vector memory

PostgreSQL + pgvector stores project embeddings for context retention.

⚡

Circuit breaker

Auto-disables agents after 3 consecutive failures. No runaway costs.

📊

JSON logging

Immutable structured logs with secret masking for audit compliance.

Meet the agents

💻

DevAgent

code

Scaffolds repos, configures CI/CD (GitHub Actions + GitLab CI), runs tests, deploys to staging.

📝

ContentAgent

content

Writes SEO content, generates images via Replicate, manages Strapi CMS editorial calendar.

📊

MarketingAgent

growth

Segments audiences, sends email campaigns (MailHog/SMTP), configures ads, generates reports.

💰

CommerceAgent

sales

Manages product catalogs, creates Stripe checkout sessions, tracks inventory, generates FAQ.

How it works

Clone & configure

git clone + cp .env.example .env — fill in your API keys.

Start the stack

make init && docker compose up — all services spin up.

Describe your goal

Send a prompt to the API or use the Next.js dashboard.

Approve & deploy

Review HITL requests, approve, and your pipeline completes.
